
# Proyecto React - Programación de Componentes

## Tecnologías utilizadas
- React
- Bootstrap
- Firebase
- React Router
- React Simple Validator

## Instalación

```bash
npm install
npm run dev
```

## Dependencias recomendadas

```bash
npm install react-router-dom bootstrap firebase react-simple-validator
```

## Funcionalidades
- Listado de productos
- Carrito de compras
- Comunicación padre-hijo
- Formularios validados
- Firebase Firestore
- Firebase Auth
- Bootstrap

## Despliegue
Compatible con Netlify.
